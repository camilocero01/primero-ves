# 📱 GUIÓN DE VENTAS - PrimeroVes
## Sistema de Páginas Web Médicas Premium

---

## 🎯 RESUMEN DEL MODELO DE NEGOCIO

**Flujo de Venta:**
1. Vendedor contacta al médico por WhatsApp
2. Presenta el servicio con demo personalizado
3. Levanta los datos del médico
4. Edita `siteConfig.ts` con información real
5. Genera slug con 3 demos listos (Elegante, Moderna, Clásica)
6. Cierra la venta mostrando el sitio funcionando

## 📋 FASE 4: GENERAR EL DEMO

### Una vez recopilados los datos:

1. Abrir `src/config/siteConfig.ts`
2. Copiar un bloque de doctor existente
3. Modificar el `slug` (ej: `dr-nombre-apellido`)
4. Llenar todos los campos con la información recopilada
5. Ejecutar `npm run build`
6. Subir a Vercel o servidor de staging

### Código de ejemplo para copiar:

```typescript
{
  slug: 'dr-nombre-apellido', // URL personalizada

  doctor: {
    nombre: 'NOMBRE COMPLETO',
    titulo: 'Dr.',
    especialidad: 'ESPECIALIDAD',
    subespecialidad: 'SUBESPECIALIDAD',
    cedulaProfesional: 'RM-XXXXX',
    foto: '/images/photo/dr-nombre.webp', // O URL de imagen
    frase: 'FRASE O LEMA',
    biografia: 'BIOGRAFÍA',
  },

  colores: {
    primario: '#1B4D6E',  // Ajustar según preferencia
    secundario: '#C9A96E',
  },

  logo: {
    url: '/images/logo-placeholder.webp',
    alt: 'Dr. NOMBRE - ESPECIALIDAD',
  },

  servicios: [
    { nombre: 'SERVICIO 1', descripcion: 'DESCRIPCIÓN', icono: 'heart-pulse' },
    { nombre: 'SERVICIO 2', descripcion: 'DESCRIPCIÓN', icono: 'activity' },
    { nombre: 'SERVICIO 3', descripcion: 'DESCRIPCIÓN', icono: 'shield' },
  ],

  horarios: [
    { dias: 'Lunes a Viernes', horas: 'HORARIO' },
    { dias: 'Sábados', horas: 'HORARIO' },
  ],

  contacto: {
    telefono: '+XX XXX XXX XXXX',
    whatsapp: 'XXXXXXXXXX', // Sin + ni espacios
    email: 'EMAIL',
    direccion: 'DIRECCIÓN COMPLETA',
    ciudad: 'CIUDAD',
    estado: 'ESTADO',
    pais: 'PAÍS',
    codigoPostal: 'CÓDIGO',
    googleMapsUrl: 'LINK GOOGLE MAPS',
    googleMapsEmbed: 'EMBED GOOGLE MAPS',
  },

  redesSociales: {
    facebook: 'URL',
    instagram: 'URL',

  },

  seo: {
```
